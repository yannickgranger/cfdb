AC-7 — compile-cost measurement (issue #105 / slice 43-B)

sccache via Redis is warm in this environment (per CLAUDE.md §10), so the
absolute numbers are lower than a cold CI runner would see. The DELTA
between default and --features git-enrich is the load-bearing signal.

$ cargo clean && time cargo build -p cfdb-cli --release
  real  0m10.553s      (default — NO git2 in dep tree)

$ cargo clean && time cargo build -p cfdb-cli --features git-enrich --release
  real  0m18.404s      (git-enrich — git2 v0.20.4 + libgit2-sys v0.18.3+1.9.2)

Delta: +7.9s on top of default.

The council's rust-systems Q1/Q6 estimate (+20-35s) assumed git2's default
features (ssh/https/ssh-key-from-memory). The feature gate here uses
`default-features = false` because enrich-git-history only reads the local
filesystem — network/crypto layers are unused. That cuts the delta below
the estimate, which is the right trade.

# Context package — Issue #213

## Issue summary
`feat(cfdb-cli): cfdb classify verb — route diff findings by debt class`. Add a diff-restricted classification entry point: take a `cfdb diff` envelope (#212, now shipped as `DiffEnvelope` in `cfdb-query`) plus `--db --keyspace` and emit `findings_by_class` limited to items whose `qname` appears in the diff's `added` ∪ `changed` sets. Implementation delegates to the shared `populate_findings_by_class` (`crates/cfdb-cli/src/scope.rs:154`); no re-implementation of classifier Cypher.
